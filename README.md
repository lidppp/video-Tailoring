# README
## 视频裁剪 
### 网站前端 + nodejs实现视频在线裁剪
### 使用技术架构
+ 前端(目前前端样式没有优化,比较丑,emmmm,有时间去好好写写样式)
  - HTML + CSS + JS + jQuery
  - 前端部分剩余浏览器窗口发生改变进度条自动适应,input的值发生改变进度条自动适应,样式优化
+ 后端
  - node.js + express + fluent-ffmpeg 
### 注意事项
> 需要本地下载 `ffmpeg` 并且配置环境变量 具体自行百度一抓一大把   

> 开发配置为限制500MB视频文件上传,可以自行再源码中修改
